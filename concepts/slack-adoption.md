# Slack 도입 및 내재화

## 개요

브랜업(주)의 Slack 도입 과정에서 발생하는 조직 문화적 과제와 대표 피드백을 추적하는 페이지.

## 현재 상태

- 코스메틱 본부: Slack으로 내부 소통 전환 중
- 외부 업체/거래처: 여전히 카카오톡 사용 → 내부(Slack)↔외부(카톡) 간극 존재
- Hermes 봇이 Slack에 상주하며 업무 자동 감지·제안 기능 제공 중

## 이상원 대표 피드백 (2026-07-13)

핵심 메시지: **"슬랙이 도움되게 계속 만들어야 가능합니다"**

| 주제 | 내용 |
|------|------|
| 전환 의지 | "저희 본부는 가능하면 이제 슬랙으로 하는데" |
| 외부 한계 | "여전히 업체는 카톡으로 되는데" |
| 체감 필요 | "무엇이 도움되는지 실제로 체감하게 매일 푸쉬해주세요" |
| 혁신 vs 관성 | "새로운 걸 계속 쓰면서 발전해야 하는데" / "확실한 게 없으면 기존 꺼로 가는 게 인간" |
| 기술 이슈 | 공유 링크 동작 불안정 (나중에 재확인 필요) |

## 과제

1. **가치 체감**: 매일의 푸쉬 알림·요약·제안을 통해 Slack 사용의 실질적 업무 효용 입증
2. **내외부 간극**: 카카오톡 의존 업체와의 소통을 Slack으로 점진적 전환 또는 연동
3. **안정성**: 링크·파일 공유의 안정적 동작 보장
4. **지속적 발전**: 새로운 기능·워크플로우를 계속 도입하여 "이게 왜 좋은지"를 반복 증명
