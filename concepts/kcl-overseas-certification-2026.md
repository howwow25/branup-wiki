# KCL 해외인증실증지원사업 (2026)

## 개요

한국건설생활환경시험연구원(KCL)이 주관하는 「2026년 인증획득제품 해외실증 지원사업」에 브랜업이 KANGEN BEAUTÉ 3-Step Facial Skincare Set으로 지원. 대면평가 통과 후 수정사업계획서 제출 및 컨설팅 진행 중.

## 사업 정보

| 항목 | 내용 |
|------|------|
| 실증사업명 | 스킨케어 기초화장품 북미 수출 |
| 실증기간 | 2026.06.01 ~ 2026.12.31 (7개월) |
| 제품 | KANGEN BEAUTÉ 3-Step Facial Skincare Set |
| 총사업비 | 70,000,000원 |
| 정부지원금 | 49,000,000원 |
| 기업부담금 | 21,000,000원 |
| 실증기관 | SGS (미국) / 엘스안전성효능연구원 |

## 실증 필요성

"등록은 진입 요건, 실증자료는 유통 요건"

FDA Product Listing 및 캐나다 CNF는 행정 등록 절차일 뿐, 실제 유통 단계에서 바이어·유통사·규제기관이 요구하는 제품 안전성·안정성·품질 검증자료는 별도로 확보해야 한다. ^[raw/articles/kcl-presentation-202607.md]

```
6월: 계획 확정
```

## 외부 자문

- **이영진 박사** (KAIST 화학공학 박사, 前 아모레퍼시픽 R&D, 現 애듀코스 대표): 처방·연구 자문
- **한상훈 박사** (성균관대 공학박사, 前 아모레퍼시픽 기술연구원 원장): 연구개발·품질관리 총괄 자문

## 결과 활용

1. **바이어 제출자료** — Enagic 본사 검토, 발주 의사결정
2. **클레임 대응자료** — 소비자·규제기관 이슈 대응 근거
3. **품질관리 자료** — 내부 QC 기준, 제조 표준 정비
4. **유통사 검토 대응** — 북미·캐나다 유통 채널 진입

## 기대 성과

- 북미 수출 신뢰도 강화
- 후속 발주 확대 (현재 80만불 → 추가 확대)
- 캐나다 및 타 국가 수출 기반 확보
- 향후 SKU 확대 기반
