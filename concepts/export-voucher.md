# 수출바우처

> 중소벤처기업진흥공단(중진공)이 중소·중견기업의 수출 역량 강화를 위해 지원하는 바우처 제도.

## 지원 개요

| 항목 | 내용 |
|------|------|
| **주관** | 중소벤처기업진흥공단 |
| **대상** | 수출 유망 중소·중견기업 |
| **지원 규모** | 기업당 최대 5천만원 ~ 1억원 (등급별 차등) |
| **자부담** | 10%~30% (기업 규모·등급별) |
| **지원 기간** | 일반 1년, 최대 3년 (성과평가 후 연장) |

## 지원 분야 (활용 가능 항목)

| 분야 | 세부 항목 | 브랜업 활용도 |
|------|----------|:----------:|
| 해외 마케팅 | 전시회 참가, 카탈로그 제작, 온라인 마케팅 | ⭐⭐⭐ |
| 수출 컨설팅 | 해외 규제·인증, 계약 검토, 시장 조사 | ⭐⭐⭐ |
| 통번역 | 문서 번역, 통역 서비스 | ⭐⭐ |
| 해외 인증 | FDA, MoCRA, CGMP, 유럽 CPNP | ⭐⭐⭐ |
| 온라인 플랫폼 | 아마존, 큐텐, 알리바바 입점 지원 | ⭐⭐ |

## 브랜업 적용 방안
- **미국 FDA/MoCRA 대응** → 해외 인증 항목 활용
- **Enagic USA 마케팅** → 해외 마케팅 바우처 (전시회, 온라인 광고)
- **계약서 검토** → 수출 컨설팅 항목 활용 가능

## 진행 상태
- ⏳ 수출바우처 공고 확인 필요 (매년 상반기·하반기)
- ⏳ 브랜업 수출바우처 활용 이력 확인 필요
