# TSonaX

> Ashton(이향석)이 개발 중인 실시간 주가 차트 분석 시스템. C#/.NET 기반.

## 개요
- **목적:** 실시간 주식 데이터 수집 → 차트 분석 → 시각화
- **기술 스택:** C# (.NET), 실시간 데이터 파이프라인
- **개발 브랜치:** `feat/minichart-tf-series-reconnect` (GitHub `howwow25/TSonaX`)
- **상태:** 마일스톤 #2 진행 중

## 주요 기능
- **미니차트 TF 시리즈:** 타임프레임별 차트 데이터 재연결 처리
- **실시간 데이터:** WebSocket 기반 실시간 시세 수신
- **분석 지표:** 이동평균, 볼린저밴드, RSI 등

## 주간 업무 우선순위
1. 🥇 TSonaX 마일스톤 #2
2. 🥈 전경원 주식지표 연구
3. 🥉 계약서·수출 관리 (아이젤·에나지크·브랜업코러스)
4. PL/IP 보험
5. 브랜업 재무 관리
6. 다우오피스 워크플로
