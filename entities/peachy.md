# Peachy — 미국 보톡스 전문 체인점

> 브랜업 전략기획에서 2026-07-10 분석한 미국 '보톡스 전문 체인점'

---

## 비즈니스 모델

Peachy는 보톡스 가격을 **정액제**로 단순화하고, 3~4개월마다 재방문하는 구독형 시술 모델:

### 1. 정액 가격
- 일반 병원: 유닛당 $12 × 40유닛 = **$480** (변동)
- Peachy: 이마·미간·눈가 통합 1회 → **$425** 정액
- 비유: "일반 병원은 고기를 무게에 따라 계산, Peachy는 정해진 가격에 한 끼 코스"

### 2. 반복 매출 구조
- 3~4개월마다 재방문 → 연 3회 방문 기준
- 고객 1명 연간 매출: **$1,275**
- 3회 시술권 선판매: **$1,125** (고객 할인 + 회사 선결제)

### 3. 고객당 매출 확대 (Upsell)
| 서비스 | 가격 |
|--------|:----:|
| 얼굴 주름 보톡스 | $425 |
| 사각턱·이갈이 시술 | $525 |
| Aquagold 피부 시술 | $375 |
| Botox/Dysport 선택 | +$150 |
| 처방 레티노이드 | $80 |
| 선크림·보습제 | $35~45 |

| 지표 | 수치 | 비고 |
|------|:----:|------|
