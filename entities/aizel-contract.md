# 아이젤 제조위탁계약

## 개요
- **계약 상대:** (주)아이젤 (경기도 화성시 동탄)
- **계약 종류:** 화장품(반제품) 제조 위·수탁 기본계약
- **갑:** 브랜업(주) — 위탁자
- **을:** (주)아이젤 — 수탁자(제조사)
- **용도:** 미국 시장(Enagic USA)을 포함한 해외 수출용
- **상태:** 아이젤 회신안 → 브랜업 수정본 1차 검토 완료 (신우 자문 반영)

## 계약 구조
- **기본계약 + 개별계약(발주서)** 이중 구조
- 개별계약(발주서)이 기본계약보다 우선함
- 개별계약 미체결 시 기본계약 규율 적용

## 주요 변경사항 요약 (아이젤 회신안 → 브랜업 수정본)

### 발주 및 납기 (제3조, 제10조)
- **발주서 발송:** 납기 90일 전 → **75일 전**으로 완화
- **납기 통보:** 사급재 입고 후 일괄 통보 → **발주서 수령 후 1차 통보** 방식으로 개선
- **납기 지연 비용:** 국내 운송비 중심 → **보관료·수출일정 변경 부대비용까지 확대**
- **지연 시 책임:** 간접손해 면책 유지 + **제3자에 실제 부담한 직접손해는 을 부담**

### 품질 및 검사 (제11조~제13조)
- **수출국 법령 검토:** 갑(브랜업) 책임이 원칙, 을(아이젤)의 참고 의견·협조 의무 추가
- **사급 원부자재 하자:** 을 전면 면책 → **을의 입고검사·보관·취급상 귀책이 있으면 책임 부담**
- **숨은 하자 조치:** 반품·정산 → **교체·시정조치까지 확대**
- **교체품 비용:** 국내 운송비 → **긴급운송비·보관료·수출일정 변경비용까지 확대**

### 품질기록 및 감사 (제16조, 제17조)
- **원료 변경:** 무단 변경 시 변경과 결함 간 객관적 인과관계 요구
- **품질기록 보존:** 유통기한 만료 후 1년 → **제조일 기준 총 6년**으로 대폭 확대
- **영업비밀 공정:** 감사 거부 → **독립 제3자 전문기관 확인** 방식으로 조정

### 제조물 책임 (제20조, 제20조의2 — 신설)
- **책임한도 신설 (제20조의2):**
  - 일반: 주문 공급가액 100% 상한
  - **예외 (상한 미적용):** 고의·중과실, IP침해, NDA위반, 인명피해, 소비자 위해, 제3자 법적배상
- **구상청구 기간:** 최종납품일 3년 → **5년**으로 확대
- **입고검사 통과 항목 을 면책 삭제**

### 대금 결제 (제23조)
- **잔금 지급:** "출고 준비 완료시" → **"갑 목적물 수령 후 10일 이내"** 로 변경
- 구매주문서 별도 약정 우선

### 리콜 (제22조)
- **비용 범위:** 대체품 제조비 한정 → **리콜공지·회수물류·시험검사·폐기·대체품제조·규제기관 대응까지 확대**
- **긴급 리콜:** 갑이 우선 진행 가능 (규제기관 명령·소비자 위해 시)

### 지식재산권 및 경업금지 (제28조, 제29조)
- **유사 배합 기준:** 성분 85% 동일 → **60% 동일**로 완화 (을의 제3자 공급 제한 강화)
- **우회거래 금지 존속:** 계약종료 후 1년 → **5년**으로 확대
- **대상 거래처 확대:** Enagic USA → **Enagic USA + Enagic Canada + 계열사·임직원·대리인·중간거래회사**
- **IP 출원 협조 의무:** 을이 갑의 IP 출원에 합리적 협조 제공

### 지체보상금 (제27조)
- **지연 1일당:** 0.1% → **0.3%** 로 상향 (상한 30%는 유지)

### 계약 종료 후 효력 (제32조)
- **잔존 조항 명시:** 비밀유지·우회거래금지·IP·품질기록·제조물책임·리콜·손해배상·분쟁해결 조항은 종료 후에도 존속

## 계약 기간
- 체결일로부터 1년, 자동연장 (1개월 전 통지 없을 시)
- 이행 중인 개별계약은 종료 후에도 완료 시까지 유효
