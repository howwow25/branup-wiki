### 2026-07-22
- **두코 감리 방식 변경** — 현장 감리 → 사진 감리로 전환 (7/21 화)
- **내일 승인 시 금요일 출고 유지** — 사진 감리 결과 이상 없으면 일정 변동 없음
- **리스크: 사진 감리 후 문제 발견 시 일정 변동 가능** — 육안 확인 불가한 부분이 있을 수 있어 최악의 경우 일정 지연 가능
- **프로세스 경고: 시안→샘플→기준협의→양산 프로세스 준수 재강조**
  - 캐나다 건 급하게 진행되며 샘플링 생략 → 양산 직행
  - 구두 기준만 있고 가시적인 샘플 기준 없는 상태에서 양산 진행
  - **앞으로 절대 지양** — 모든 단계 반드시 준수
