## 누적 지식 (Knowledge Base)

### 인쇄 검토 프로세스
1. **AI 원본 제작** → Adobe Illustrator 최종 디자인 원본
2. **PDF 검수본 출력** → 변경 사항 확인 (동박 위치, 글자 추가/변경 등)
3. **최종 승인** → `[확정]` prefix 파일 등록
4. **인쇄소 전송** → 후가공 사양, 감리 일정 확정
5. **주의사항:** 인쇄 문안 변경 시 최종 후가공 사양·감리 일정 변경·다수 원본 파일 수정 필요

### 디자인 파일 명명 규칙
- `[확정]` prefix: 최종 승인 완료 파일
- `날짜-버전` suffix: `0714` = 7월 14일자, `15300ea` = 15,300매 수량
- 국가명 포함: `미국-`, `캐나다-`
- 파일 유형 구분: `-표지-`, `-전면`, `-면지`, `-인쇄`, `-고시사항`

### 색상 사양
- **PANTONE 7570 C** — 브랜업 코퍼레이트 컬러 (표지 디자인 적용)
- **용기 색상:** [맞춰찍기] — 인쇄 시 용기 색상에 맞춰 인쇄
- **증착 후가공:** JNJ 자석&명판 타입에 적용 (2026-07-15 기준 샘플 지연)
